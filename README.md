# SAKTI — Intelligent Tailor Allocation & Cooperative Management System

> 2nd Place — PCU × SUTD GEO 2026

SAKTI is a hybrid decision-support and cooperative operations prototype
developed with Koperasi Sumber Mulia Barokah.

The platform helps cooperative administrators manage projects, inventory,
suppliers, purchases, tailor assignments, production progress, and workload
distribution through one integrated Streamlit dashboard.

Its allocation workflow combines:

1. ML-assisted tailor profiling based on historical performance attributes.
2. Context-aware weighted scoring based on urgency, quantity, speed,
   neatness, specialization, and distance.
3. Real-time workload balancing using current assignments and availability.

The final recommendation remains human-reviewed and can be overridden by
the cooperative administrator.
